Данный проект представляет собой шифровальщик\дешифровальщик шифром Виженера с простым и интуитивно понятным графическим интерфейсом.
Данный проект состоит из двух частей: Verenchuk - непосредственно сам проект, UnitTestProject - тесты.
Реализована возможность открывать любой файл на дисках, использую стандартный менеджер файлов.
Реализована возможность задавать ключ.
Реализована возможность сохранять файл без шифрования.
Реализована возможность редактировать текст средствами программы.
